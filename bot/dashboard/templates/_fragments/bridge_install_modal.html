Install Telegram Bridge

Animaya connects to one Telegram bot you control. To finish installing the bridge, paste a bot token issued by @BotFather.

  1. Open @BotFather in Telegram.
  2. Send /newbot and follow the prompts (name + username).
  3. BotFather replies with a line like Use this token to access the HTTP API: 12345:ABC… — copy it.
  4. Paste the token below. It is validated via getMe before anything is saved.

Stored locally in this LXC only. Never logged; never returned by the API.
